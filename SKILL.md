# 发票归档 Skill

按财务凭证 Excel 表格，将发票 PDF/XML 文件按「记账期间 → 凭证字号」两级目录自动归档。

## 触发条件

用户说：归档发票、发票归档、按凭证字号归档、整理发票 等

## 工作流程

1. 确认用户提供了 Excel 文件路径和发票所在目录
2. 运行 `archive_invoices.py` 脚本
3. 汇报归档结果

## 脚本说明

`archive_invoices.py` 功能：
- **自动检测表头列位置** — 适配不同财务系统导出的 Excel（列布局可能不同）
- **按记账期间建大文件夹** — 如 `2026年01期/`
- **按凭证字号建小文件夹** — 省略前6位数字前缀，如 `202601 转-11` → `转-11/`
- **增量归档** — 已存在的文件夹复用，不删除已有文件
- **智能匹配** — 优先 H 列精确匹配，备选 C 列发票号码模糊匹配
- **空文件夹不创建** — 找不到发票文件的凭证字号不会创建空目录
- **去重检查** — 目标位置已有同名文件时跳过

## 调用方式

```bash
python archive_invoices.py
```

脚本内修改 `EXCEL_PATH` 和 `INVOICE_DIR` 即可适配不同批次。

## 依赖

- Python 3.x
- openpyxl (`pip install openpyxl`)
